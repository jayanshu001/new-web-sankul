6c · Admin — Chat no session

Posts as admin via POST /admin/live-chat/message — broadcast to the room over Socket.IO, so a customer tab joined to the same session sees it instantly in the right-hand panel. Customer messages appear here on refresh (admins aren't on the socket).

Select a live session in 6b.

6d · Admin — Polls no session

Create a poll (POST /admin/live-polls) — pushed live to the room; customers vote from the right-hand panel and counts stream back here on refresh. Creating a new poll auto-closes the previous active one.

Select a live session in 6b.

6e · Admin — Go live from camera

Broadcast your browser camera + mic straight to the session selected in 6b — it must be CREATED (live). The browser streams WebM chunks over a WebSocket; the server runs ffmpeg to transcode and push RTMP to Streamos, so no OBS is needed. Chrome recommended; open this page on localhost (camera access needs a secure context).

Flow: section 6 → Go Live now (creates the stream) → 6e → Enable camera → Start broadcast. Customers watching in section 3 see your camera within a few seconds (HLS latency).
